How to Use

Type or paste Markdown on the left; the rendered result appears on the right.

  • Math: use $inline$, $$display$$, \(…\) or \[…\].
  • Tables: click Table to insert, or place the cursor in a table and use Edit Table from the right-click menu.
  • Lists: select lines, then use the list button or right-click → Bullet / Number.
  • Undo / Redo: Cmd/Ctrl+Z and Cmd+Shift+Z / Ctrl+Y.
  • Import / Export: drag a .md file anywhere, or use the Import / Export panel to copy and download.
  • History: each Clear saves a snapshot you can reload from the History sidebar.
